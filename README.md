# Retencion-y-cancelacion-de-membresias-

Este proyecto busca identificar los factores asociados al abandono de membresías y segmentar a los clientes según su nivel de riesgo.

## Contenido

-Modelos de clasificación: Regresión logística y Random Forest
-Segmentación de clientes: K-means y clustering jerárquico
-Identificación de factores clave en la retención
-Interpretación de clústeres de riesgo y lealtad
